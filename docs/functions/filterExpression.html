フィルタ式

teigi の Lodget 内でフィルタ式を使用する場合があります。

  • イベント定義でオブジェクトの変更がイベントの対象フィルタ、発火条件フィルタ
  • インタフェース定義のデフォルトの絞り込みフィルタ（＝ACLにおける対象フィルタ）

フィルタ式は基本的に Mongo のフィルタ式に準じており、フィルタ式エディタで編集できます。 フィルタ式の中の属性参照はグローバルバリデーションチェックの対象となります。
